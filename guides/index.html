The deep guides

  • Discord Emoji Size — Discord emoji size is 128×128 pixels, max 256KB, PNG or GIF.
  • Twitch Emote Size — Twitch emotes need three sizes: 28×28, 56×56 and 112×112 pixels, PNG with transparency, under 1MB.

More platforms get their own guides as we go — the table above is always the complete picture.
