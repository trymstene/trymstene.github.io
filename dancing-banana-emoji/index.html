Quick setup

Add it to Discord

  • Grab the emoji file: Discord emoji (128px GIF) ↓
  • In Discord: Server Settings → Emoji → Upload Emoji and pick the file.
  • Name it :bananadance: and save. Done.
  • Good to know: animated server emoji show for everyone, but members need Nitro to send them — the still PNG makes a static one anybody can use.

Quick setup

Add it to Slack

  • Grab the emoji file: Slack emoji (128px GIF) ↓ — 13KB, comfortably under Slack's size cap.
  • In any message field, click the smiley → Add emoji → Upload image.
  • Name it :bananadance: — it animates right in the message line.

Quick setup

Add it to Telegram

  • Easiest: send the 512px GIF ↓ in any chat, tap it and choose Save to GIFs — it lives in your GIF drawer forever.
  • Building a real sticker pack? Telegram's @Stickers bot takes static 512×512 PNGs: sticker PNG (512px) ↓
  • Message @Stickers → /newpack → send the PNG → pick an emoji for it.

Quick setup

Add it to Twitch

  • Affiliates & partners: grab the emote file (128px GIF) ↓
  • Creator Dashboard → Viewer Rewards → Emotes → upload as an animated emote — Twitch auto-generates the 112/56/28px sizes.
  • Prefer a static emote? Use the still PNG the same way.

Quick setup

Everywhere else

  • Microsoft Teams: emoji picker → + Add emoji → upload the 128px GIF (if your org allows custom emoji).
  • WhatsApp / Signal: feed the 512px PNG to any sticker-maker app — it's already square and transparent.
  • Anywhere that takes GIFs (iMessage, forums, docs, presentations): the 500px transparent GIF just works.

Is there a dancing banana emoji?

Is there a dancing banana emoji to copy and paste?

Not as a Unicode text emoji you can type like 🍌 — the dancing banana is an animated image. So you add it as a custom emoji/sticker: grab the free file above and upload it to Discord, Slack, Telegram and the rest.

Download the 128px emoji GIF, then Server Settings → Emoji → Upload Emoji → pick the file. It's pre-cropped — no editing needed.
