Quick setup

How to add it to Discord

  • Download the transparent banana above (GIF for animated, PNG for still).
  • In Discord: Server Settings → Emoji (or Stickers) → Upload.
  • Pick the file, give it a name like :banana:, and save. Done. 🍌

On Slack, add it as a custom emoji; on Telegram, add it as a sticker. Same file works everywhere.
